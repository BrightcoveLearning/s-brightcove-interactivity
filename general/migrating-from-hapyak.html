Folders can be created inside of the Interactivity module and can be used to organize projects, templates and experiences. Folders are displayed in the side navigation panel. The number of projects/templates/experiences in each folder is displayed next to the folder name.

Note: Nested folders are not supported

Creating folders

To create a new folder, follow these steps:

  1. Open the Interactivity module.
  2. In the side navigation panel, expand the Projects, Templates, or Experiences section.
  3. Click Create folder in the appropriate section.
  4. Enter a name for the folder using the following guidelines. Press enter.
    • Folder name is limited to 100 characters
    • Folder names must be unique

Adding items to folders

There are a couple of ways to add projects/templates/experiences to a folder. You can:

  • Add a single asset to a folder
  • Adding multiple assets a folder
Note: There is no limit to the number of projects/templates/experiences that can be in a folder.

Adding a project/template/experience to a folder

Following these steps to add a project/template/experience to a folder.

  1. Open the Interactivity module.
  2. Click the action button (...) next to the project/template/experience and then click Move to Folder.
  3. Select a folder and click Move.

Adding multiple projects/templates/experiences to a folder

Multiple projects/templates/experiences can be added to a folder by following these steps.

  1. Open the Interactivity module.
  2. Click the checkboxes next to the projects/templates/experiences to move. The action bar will appear at the bottom of the page.
  3. Click Move to Folder.
  4. Select a folder and click Move.
