The Style Editor provides an easy to use interface that can be used to style annotations:

  • Chapter menus
  • Overlays
  • Hotspots
  • Quizzes

After the styling is complete, the changes can be published so that the styling is applied to all annotations in the account.

Note: For information on styling annotations, see Styling Annotations.

To access the Style Builder, follow these steps:

  1. Open the Interactivity module.
  2. Click the settings icon () at the top of the page.
  3. In the primary navigation, click Style Editor.

By default, when the style editor opens, the chapter menu styling options are displayed. Use the left navigation to access the styling options for all of the annotation types.

  • Chapter Menu
  • Overlay
  • Hotspot
  • Quiz

After styling changes are complete, click Publish to save the changes. Click Restore to revert back to the default styling settings.

Styling chapter menus

To style chapter menus, expand the Chapter Menu option in the left navigation. The following styling options are available:

  • Title Background Color - Chapter menu title background color
  • Title Font Color - Chapter menu title font color
  • Body Font Color - Chapter listing font color
  • Body Hover Font Color - Chapter listing font color on hover
Note: Additional chapter menu settings are available as part of the Config settings.

To select a color value, follow these steps:

  1. Click on the color circle to open the color picker.
  2. Use the color slider to set a color range.
  3. Use the opacity slider to set the opacity for the color.
  4. Click in the color square to select a color.
  5. Click the X to close the color picker.

As styling changes are made, they will be reflected in the Live Preview player on the page.

Styling overlays

To style chapter menus, expand the Overlay option in the left navigation. The following styling options are available:

  • Background colors, font colors and border thickness for overlays

Styling hotspots

To style hotspots, expand the Hotspot option in the left navigation. The following styling options are available:

  • Background color for hotspots

Styling quizzes

To style quizzes, expand the Quiz option in the left navigation. The following styling options are available:

  • Background, font and button colors for quizzes

Publishing changes

After styling changes are complete, click Publish to save the changes. Click Restore to revert back to the default styling settings.
