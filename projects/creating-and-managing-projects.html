A Project is a single video with annotations. Annotations may include links, images, hotpots, quizzes, etc. that appear during video playback.

Automatically creating a project

The Interactivity plugin can be configured with an option so that Brightcove videos that are played using an Interactivity-enabled player will automatically create Interactivity projects.

Note: If the plugin options are not configured, projects will not be auto-created.

When a video is played inside of an Interactivity-enabled Brightcove Player, the following things happen:

  • The Brightcove video ID, title, tags and custom fields are passed to Brightcove Interactivity
  • Brightcove Interactivity attempts to look up the video ID within your group
  • If no matching record is found, then Brightcove Interactivity creates a project with the data that was sent

The following steps will automatically create an Interactivity project. Make sure you have an Interactivity-enabled player before continuing:

  1. Login to Video Cloud Studio.
  2. Open the Media module.
  3. Select a video to publish by checking the box next to the video thumbnail.
  4. Click Publish and embed... Ë‡ > Publish to web.
  5. Select a player with the Interactivity plugin.
  6. Click the preview link to open a preview in a new browser tab.
  7. Play the preview video (the video only needs to be played for a few seconds). This will auto-create a project in Brightcove Interactivity.
    Note: You must preview the video using the preview link. Previewing the video with the preview player in the Media module will not auto-create a project.
  8. Open the Interactivity module. A list of projects will appear.
  9. Confirm the video you just published appears in the project list.

Manually creating a project

Projects can be manually created using the Interactivity module. Follow these steps to manually create a project.

  1. Login to Video Cloud Studio.
  2. Open the Interactivity module.
  3. Click New Project.
  4. Enter a Project Title.
  5. Select a video to use in the project.
  6. Click Create. The project will appear in the Projects listing.

Editing a project

Follow these steps to edit a project.

  1. Open the Interactivity module.
  2. Click on a project Title or click the action menu (...) associated with the project and click View.

Moving a project to a folder

Folders can be created and used to organize projects. You must have created folders before you can move a project to a folder. Follow these steps to move a project to a folder.

  1. Open the Interactivity module.
  2. Click the action menu (...) associated with the project and click Move to Folder.
For more information on folders, see Working with Folders.

Duplicating a project

Duplicating a project is a great way to save time when creating new projects that are similar to ones already created. Simply change the video source of the new, cloned project and adjust the annotations as needed.

Follow these steps to duplicate a project.

  1. Open the Interactivity module.
  2. Click the action menu (...) associated with the project and click Duplicate.
  3. Click Duplicate.

This will create an exact clone of the project including all annotations, tags, and custom CSS, titled "{project_name} - Copy".

Deleting projects

To delete a project, find the project on the project list, click the action menu (...) associated with the project and then click Delete.

Converting a project to a template

Converting a project to a template will permanently remove the selected project from the Projects section and turn it into a template. This action cannot be undone. Take note of which projects you convert or make sure to make copies of projects before you convert them.

Follow these steps to convert a project to a template.

  1. Open the Interactivity module.
  2. Click a project Title or click the action menu (...) associated with the project and click View.
  3. Click Convert to template.
  4. Click Confirm.

After the project is converted it will appear in the Templates section.

Configuring project settings

The project settings can be used to edit the project title, add tags, add templates, assign a custom CSS file and replace the video associated with a project.

Follow these steps to access the project settings.

  1. Open the Interactivity module.
  2. Click a project Title or click the action menu (...) associated with the project and click View.
  3. Click Settings.

For more information, see Configuring Project Settings.

Embedding a project

Embedding a project generates the embed code needed to embed the project on a website.

Follow these steps to generate the embed code.

  1. Open the Interactivity module.
  2. Click a project Title or click the action menu (...) associated with the project and click View.
  3. Click Embed.

For more information, see Generating Embed Code.
