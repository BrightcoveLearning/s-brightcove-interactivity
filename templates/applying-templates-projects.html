A template is a track (or group) of annotations that can be applied to multiple projects. Templates can be used to quickly apply identical annotations to multiple videos with one click.

Multiple template tracks can be added to a project. If the template track is updated, Brightcove Interactivity will automatically update all projects using the template.

Applying templates using tags

Tags can be added to templates as a way to quickly add annotations to videos that are published using an Interactivity-enabled player. If a published video has the same tag that was assigned to the template, the template annotations will be applied. This is a quick and easy way to add the same annotations to dozens or hundreds of videos.

Follow these steps to use a tag to apply a template to a published video.

  1. Login to Video Cloud Studio.
  2. Open the Interactivity module.
  3. In the left navigation, click Templates > All templates.
  4. Click a template Title or click the action menu (...) associated with the template and click View.
  5. Click Settings.
  6. Locate the Apply Templates to Projects by Tag field and click it.
  7. Add one or more Tags. Press enter after each tag.
  8. Click Save.

By adding the tag company_logo to the template, any video that is tagged company_logo and published using an Interactivity-enabled player will have the template annotations applied. Follow these steps to tag and publish a sample Video Cloud video.

Please note that if a video has already been added to a project, it is not possible to generate a new project by adding the template tag to the video using the media module.
  1. Login to Video Cloud Studio.
  2. Open the Media module.
  3. Click on a video Name to open the video properties page.
  4. In the Video Information panel, click Edit.
  5. Add the same tag that was used to tag the template.
  6. Click Save.
  7. Publish the video using an Interactivity-enabled player.
    Note: You must enable the Interactivity plugin for the Brightcove Player to create projects when adding a tag to the video. For information on creating an Interactivity-enabled player, see Configuring a Brightcove Player with the Interactivity Plugin.
  8. Play the video and confirm that the annotation(s) appear.

Applying templates to all projects

Adds the template to every project on the account, regardless of tags.

Follow these steps to apply a template to all projects on the account.

  1. Login to Video Cloud Studio.
  2. Open the Interactivity module.
  3. In the left navigation, click Templates > All templates.
  4. Click a template Title or click the action menu (...) associated with the template and click View.
  5. Click Settings.
  6. Locate the Apply to all projects switch.
  7. Turn on Apply to all projects switch.
  8. Click Save.

Manually applying a template to a project

It is also possible to manually apply one or more templates to a project. To manually apply a template to a project, follow these steps:

  1. Login to Video Cloud Studio.
  2. Open the Interactivity module.
  3. Click a project Title or click the action menu (...) associated with the project and click View.
  4. Click Settings.
  5. Click Add Templates.
  6. Select one or more templates to apply to the project.
  7. Click Apply.
  8. Confirm that the selected templates appear on the project settings dialog.
  9. Click Save.

Removing a template from a project

To remove a template from a project, follow these steps:

  1. Open the Interactivity module.
  2. Click a project Title or click the action menu (...) associated with the project and click View.
  3. Click Settings.
  4. Hover over the template and click the remove icon.
  5. Click Save.
