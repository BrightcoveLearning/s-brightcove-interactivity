Brightcove Interactivity is a flexible and extensible platform for creating different types of interactive videos. There are 3 core types of interactive videos

  • Projects
  • Templates
  • Experiences

Projects

A Project is a single video with annotations. This is the most frequently used video type. For information on projects, see Creating and Managing Projects.

Templates

A Template is a set of annotations designed to be added many videos. For information on templates, see Creating and Managing Templates.

Experiences

An Experience is a Project combined with other elements (like synced slides or an e-commerce cart) outside the video player that dynamically interact with player. For information on experiences, see Overview of Experiences.
