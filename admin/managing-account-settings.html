To access the Admin settings, click Admin in the primary navigation.

Note: You must have a role of admin in order to access the Admin menu.

The following settings are available:

  • Group name
  • Group Members
  • Sub Groups
  • Storage Used
  • Brightcove Support

Group name

The Group Name section displays the name of the current group. To edit the name, enter a new name and press enter. The new name will be reflected in the group selector at the top of the page immediately.

Group Members

The Group Members option allows you to manage the group members who have access to your Brightcove Interactivity account. For more information on managing group members, see Managing Group Members.

Sub Groups

The Sub-Group option allows you to manage the sub-groups in your Brightcove Interactivity account. For more information on managing sub-groups, see Managing Sub-Groups.

Storage Used

The Storage Used section will display the amount of storage used by your Brightcove Interactivity account.

Brightcove Support

The Brightcove Support section has helpful links to documentation and to contact Brightcove Support.
