The Interactivity plugin for the Brightcove Player allows the player to take advantage of the features provided by Brightcove Interactivity. When annotations are added to an Interactivity project, they will automatically appear on the corresponding video.

Before you get started

Before you can started with video interactivity, you must have a Brightcove Player configured with the Interactivity plugin. Adding the Interactivity plugin to a Brightcove Player will allow annotations to be displayed inside of the player during video playback. The Interactivity plugin can coexist with other player plugins. For complete steps on creating an Interactivity-enabled Brightcove Player, see Configuring a Brightcove Player with the Interactivity plugin.

Creating an Interactivity project

When a video is played inside of an Interactivity-enabled Brightcove Player, the following things happen:

  • The Brightcove video ID, title and tags are passed to Brightcove Interactivity
  • Brightcove Interactivity attempts to look up the video ID within any existing projects
  • If no matching record is found, then Brightcove Interactivity creates a project with the data that was sent

The following steps will automatically create an Interactivity project:

  1. Login to Video Cloud Studio.
  2. Open the Media module.
  3. Select a video to publish by checking the box next to the video thumbnail.
  4. Click Publish and embed... ˇ > Publish to web.
  5. Select an Interactivity-enabled player.
  6. Click the preview link to open a preview in a new browser tab.
  7. Play the preview video (the video only needs to be played for a few seconds). This will auto-create an Interactivity project.
    Note: You must preview the video using the preview link. Previewing the video with the preview player in the Media module will not auto-create a project.
  8. Open the Interactivity module. A list of projects will appear.
  9. Confirm the video you just published appears in the project list.

Adding a text overlay

In this section a text overlay will be added just to validate that the interactivity can be viewed inside of a Brightcove Player.

  1. Click on the project Title to open it.
  2. In the bottom toolbar, click on the text overlay icon.
  3. Click on the Editor link in the menu.
  4. Enter some sample text for the overlay and click Submit.
  5. Click Done.

For information on adding text and image overlays to videos, see one of the following:

  • Adding a Text Overlay to a Video
  • Adding an Image Overlay to a Video
  • Adding a Transparent Overlay to a Video

Previewing and generating embed code for a video

Once an interaction has been added to a project, preview the video to ensure the interactivity is working as expected. When previewing and generating the embed code in the Interactivity module, the project will use the Interactivity-enabled player that is configured as part of the account defaults. The project can also be previewed and published inside of the Media module.

Follow these steps to preview and generate the embed code for the project:

  1. Click Embed.
  2. Click Preview to preview the project in another browser tab.
  3. Confirm that the interaction appears.
    Note: By default, the text overlay is configured to display for 3 seconds.
  4. Copy the embed code and paste it into your web page.

Previewing and generating embed code using the Media module

Projects can also be previewed and published using the Media module. Follow these steps to preview and generate the embed code using the Media module:

  1. In the primary navigation, open the Media module.
  2. Select the video that the interactivity was added to by clicking the checkbox next to the video thumbnail.
  3. Click Publish and embed... ˇ > Publish to web.
  4. Select an Interactivity-enabled Brightcove Player.
  5. Click the preview link to preview the video.
  6. Copy the embed code and paste it into your web page.
