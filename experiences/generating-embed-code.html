In order to display an experience inside of a web page or application, embed code must be generated. Follow these steps to generate the embed code for an experience.

  1. Open the Interactivity module.
  2. Click a project Title or click the action menu (...) associated with the project and click View.
  3. Click Embed.
  4. (Optional) Click Get URL to get a URL to the project that can be shared with others.
  5. (Optional) Click Preview to open a preview of the project in another browser tab.
  6. Select the embed code to use.
    • Standard (iframe) - Recommended for most cases. Places the player inside an <iframe> tag. If player dimensions are specified, the iframe will be sized to the player dimensions. If no player dimensions are specified, the iframe can be placed inside of a <div> tag and the responsive nature of the player will resize the player to occupy the entire space of the <div>.
    • Advanced (JavaScript) - Allows the player to exist in the the HTML page directly, not in an iframe. This makes it easier for developers to access the player and associated properties and events. By using the advanced embed code, you lose the ease of use of the player as a single, contained unit in social media applications and sharing.
  7. Click Done.
