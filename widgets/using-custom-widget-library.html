The custom widget library is a set of advanced interactions that you can add with one click to projects. There is no need to write any HTML, JavaScript or CSS code. All annotations should be thoroughly tested before going live.

Adding a custom widget

Follow these steps to add a custom widget to a project:

  1. Login to Video Cloud Studio.
  2. Open the Interactivity module.
  3. Click on a project Title to open it.
  4. In the bottom toolbar, click on the widget icon.
  5. Click on a widget.
  6. Click ADD. The widget is added on top of the project video.
  7. Click and drag the widget to where you would like the text to appear over the video.

Review the documentation for information on configuring the widget:

  • Poll widget
  • Custom Form widget
  • Sign-In widget
