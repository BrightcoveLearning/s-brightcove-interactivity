The User Sentiment widget can be used to gather video feedback by letting viewers rate their experience. During video playback, a rating widget will appear over the video. Different display options are available:

  • Thumbs Up/Thumbs Down
  • Star Rating (out of 5)
  • Emoticons

The text above the rating can be easily changed and the widget can be positioned anywhere on the player (default is upper right corner). When the icon is clicked, it stays in the clicked/selected state for the remainder of the video. A User Sentiment report is available to provide the aggregate feedback about the ratings.

User like counts

When using the Thumbs icon type, you can optionally display the counts for the number of likes and dislikes.

The two counts can be activated separately from the configure modal. A threshold can be set for each count which will prevent the count from displaying until a certain number of likes/dislikes have been recorded.

The like and dislike counts do not have to be displayed. Brightcove Interactivity will still continue to collect the data in case you decide to display them at a later date.

Adding a user sentiment widget

Typically, the User Sentiment widget is added to a template so that it can be easily added to multiple videos. The steps in this topic will create a new template and then add the user sentiment widget to the template.

  1. Login to Video Cloud Studio.
  2. Open the Interactivity module.
  3. In the left navigation, click Templates > All templates.
  4. Click New Template.
  5. Enter a Template Title and click Create.
  6. Confirm that the template is created. A sample video is automatically loaded into the template. To change the video, click Replace Video and select a new video.
  7. In the bottom toolbar, click on the widget icon.
  8. Click the User Sentiment widget.
  9. Click ADD. The widget is added on top of the sample video.
  10. Hover over the widget and click CONFIGURE.
  11. Enter a USER PROMPT. This is the text that will appear at the top of the widget.
  12. Select the TYPE of widget.
  13. Choose to display the COUNT SETTINGS (displayed only for Thumbs type). A threshold can be set for each count which will prevent the count from displaying until a certain number of likes/dislikes have been recorded.
  14. Set the icon/font and background color settings as needed.
  15. The DISPLAY SETTINGS control how the widget appears during video playback.
    • Beginning - Controls how the widget will appear when the video starts playing
    • Middle - Controls how the widget will appear as the video is playing
    • End - Controls how the widget will appear at the end of the video

    The following example would display the sentiment widget for 5 seconds when the video starts playing, then it would minimize and then expand for the last 5 seconds of the video.

  16. Click SAVE.

To quickly add the User Sentiment widget to videos, add a tag to the template. If a video is published using an Interactivity-enabled Brightcove Player and if the video has the same tag that was assigned to the template, the User Sentiment widget will be displayed during video playback.

Follow these steps to add a tag to the template.

  1. At the top of the page, click Settings.
  2. Locate the Tags field and click it.
  3. Add one or more Tags. This example is using a tag of sentiment.
  4. Click Save.

By adding the tag sentiment to the template, any video that is tagged sentiment and published using an Interactivity-enabled player will have the User Sentiment widget displayed during video playback.

Note: There may be a delay of up to one hour before the User Sentiment widget will appear in published Video Cloud videos.

Reviewing sentiment reports

The User Sentiment widget will collect a "rating" when the viewer clicks on a satisfaction emoji. Since there are three types of emojis which have been implemented (thumbs, stars and a smileys), each has its own rating map which is then normalized into a score from 1-5.

The User Sentiment Summary report provides information on all projects with the User Sentiment widget. To access the report, follow these steps:

  1. Login to Video Cloud Studio.
  2. Open the Analytics module.
  3. In the left navigation, locate the Interactivity section and click Core Reports.
  4. Select a Date Range for the report.
  5. Use the report selector to select the User Sentiment Summary report.
  6. Click Run.
Note: Click Advanced to specify specific Project IDs or Experience IDs.

The report will display all projects with the User Sentiment widget.
