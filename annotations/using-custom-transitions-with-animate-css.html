Animate.cssÂ is a library of ready-to-use, cross-browser animations for use in web projects. Animate.css was created by Daniel Eden (GitHub ). Animate.css can be used to enhance how annotations appear during video playback.

Follow these steps to add the Animate.css transitions to annotations.

  1. Login to Video Cloud Studio.
  2. Open the Interactivity module.
  3. Click on a project Title to open it.
  4. Click Settings.
  5. In the Interactive Projects CSS (URL) field, add this URL:
    //hapyak_demos.s3.amazonaws.com/css/light-styles.css
  6. Click Save.
  7. To add animations to annotations, click the Edit menu for an annotation.
  8. Click More...
  9. Click Add Class.
  10. In the Classes field, enter animated <the name of the animation>, for example animated zoomInLeft.
  11. Click Submit.
  12. Click Done.
Note: Click Preview in the annotation menu to test the transition.
