Introduction

 

The Interactivity module provides the ability to add chapters to videos. Chapters make it easier for viewers to go to a specific point in a video. This is especially helpful for longer form videos. Rather than scrubbing through the video, the chapter menu can be used to jump to a specific point in the video. There are two elements needed to add chapters:

  • Bookmarks - Identify specific times in the video
  • Chapter Menu - lists all of the bookmarks
Tip: Name chapters with descriptive titles that will help your users identify content as well as allow you to make sense of the reports.

Working Sample

Before you get started

Before you can add interactivity to a player, you need an Interactivity-enabled Brightcove Player and you need to make sure an Interactivity project has been created. For information on how to do this, see Getting Started with Video Interactivity.

Adding a chapter menu

To add a chapter menu, follow these steps:

  1. Login to Video Cloud Studio.
  2. Open the Interactivity module.
  3. Create a new project or click on a project Title to open it.
  4. In the bottom toolbar, click on the chapter menu icon.
  5. To change the title of the Chapter Menu, click on it. If the chapter title is deleted, a hamburger menu will be displayed.
  6. Click Done.
Note: The Style Builder can be used to easily style the chapter menu. For information, see Using the Style Editor.

There are several annotation options available that control the appearance of the chapter menu. Click Edit > More... to expand the menu.

Styles

The Styles setting controls how the chapter menu displays

  • Fullscreen - The chapter list will fill the player
  • Column - Displays the chapters in a single column

Initial State

The Initial State setting controls how the chapter menu displays when it first appears:

  • expanded - Menu appears expanded
  • collapsed - Menu appears collapsed

Position

The Position setting controls the cardinal corner of the chapters menu:

  • top-left
  • top-right
  • bottom-left
  • bottom-right
Note: Brightcove recommends choosing either top-left or top-right in order to be able to easily navigate into the control bar.
Note: For information on the other annotation settings, see Annotation Options.

Adding chapters

To add a new chapters, follow these steps:

  1. Play or scrub through the video. When you get to the point where a chapter needs to be added, click the chapter marker icon.
  2. Enter a chapter name and press enter. The name will be shown to the viewer in the chapter menu.
    Note: In order to ensure that chapter data passes properly to analytics, Brightcove recommends giving each chapter a unique name.
    Note: If a chapter name is not entered, Brightcove Interactivity will assign a name of "Chapter #" with the # corresponding to where the chapter appears in the list of all the chapters.
    Note: The chapter name will only be displayed in the chapter menu, it will not be displayed during video playback.
  3. Continue playing the video and add chapters at each major topic or section of the video.
  4. Click the chapter menu and confirm that all chapters appear in the chapter menu.
  5. Click Embed.
  6. Click Preview. Confirm that when you click on a chapter, that part of the video plays.

Reviewing chapter data

The Chapter Summary in the Core reports can be used to view the number of views and clicks on chapters. For information, see the Core Reports Reference topic.
